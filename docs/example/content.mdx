# MDX Features Guide

This document showcases all available MDX features and components.

## 1. Basic Text Formatting

Regular paragraph text with **bold**, *italic*, and ***bold italic*** formatting.

You can also use `inline code` and [links](https://example.com).

## 2. Lists

### Unordered Lists
- First item
- Second item
  - Nested item 1
  - Nested item 2
- Third item

### Ordered Lists
1. First step
2. Second step
   1. Nested step 1
   2. Nested step 2
3. Third step

### Mixed Lists
1. Main point
   - Supporting detail
   - Another detail
2. Second point
   - More details
     1. Sub-point
     2. Another sub-point

## 10. Additional Markdown Features

### Task Lists
- [x] Completed task
- [ ] Incomplete task
  - [x] Nested completed task
  - [ ] Nested incomplete task
- [ ] Another task

### Footnotes
Here's a sentence with a footnote[^1].

And another sentence with a footnote[^2].

[^1]: This is the first footnote.
[^2]: This is the second footnote with multiple lines.
    Indent the lines to include them in the footnote.

### Definition Lists
First Term
: This is the definition of the first term.

Second Term
: This is the definition of the second term.
: Can have multiple definitions.

### Keyboard Shortcuts
Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy.
